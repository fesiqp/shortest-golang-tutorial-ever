# Shortest Golang Tutorial Ever

## Getting Started

Clone this repository
```
$ git clone https://github.com/fesiqp/shortest-golang-tutorial-ever
```

Enter the directory
```
$ cd shortest-golang-tutorial-ever
```

Run the example:
```
$ go run main.go
```
